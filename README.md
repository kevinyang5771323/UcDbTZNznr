# 医院后台管理系统 java466

## 项目概述

医院后台管理系统是基于Django框架开发的一套后台管理系统。该系统支持二次开发，适用于学生和在职人员学习与使用，旨在提高医院管理的效率。

## 技术栈

- 后端：Django
- 数据库：SQLite
- 编程语言：Python
- 前端：HTML

## 功能模块

### 系统角色

- 管理员

#### 功能页面

- 登录页面

- 医生管理页面
- 患者管理页面
- 病房管理页面
- 科室管理页面
- 药品管理页面

#### 基础操作

- 增删改查

## 运行环境

- Python环境
- Django框架
- SQLite数据库

1. 准备Python运行环境
